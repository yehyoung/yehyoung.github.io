SelectMany<TSource,TResult>(IEnumerable, Func<TSource,IEnumerable>) 將序列的每個項目都投影成 IEnumerable，並將產生的序列簡化成單一序列。 SelectMany<TSource,TResult>(IEnumerable, Func<TSource,Int32,IEnumerable>) 將序列的每個項目都投影成 IEnumerable，並將產生的序列簡化成單一序列。 各來源項目的索引是在該項目的投影表單中使用。 SelectMany<TSource,TCollection,TResult>(IEnumerable, Func<TSource,IEnumerable>, Func<TSource,TCollection,TResult>) 將序列的每個項目投影為 IEnumerable、將產生的序列簡化成單一序列，並對其中的每個項目叫用結果選取器函式。 SelectMany<TSource,TCollection,TResult>(IEnumerable, Func<TSource,Int32,IEnumerable>, Func<TSource,TCollection,TResult>) 將序列的每個項目投影為 IEnumerable、將產生的序列簡化成單一序列，並對其中的每個項目叫用結果選取器函式。 各來源項目的索引是在該項目的中繼投影表單中使用。

SequenceEqual(IEnumerable, IEnumerable) 使用項目之型別的預設相等比較子來比較項目，以判斷兩個序列是否相等。 SequenceEqual(IEnumerable, IEnumerable, IEqualityComparer) 使用指定的 IEqualityComparer 來比較項目，以判斷兩個序列是否相等。

Single(IEnumerable) 傳回序列的唯一一個元素，如果序列中不是正好一個元素，則擲回例外狀況。 Single(IEnumerable, Func<TSource,Boolean>) 傳回序列中符合指定之條件的唯一一個元素，如果有一個以上這類元素，則擲回例外狀況。

SingleOrDefault(IEnumerable) 傳回序列的唯一一個項目，如果序列是空白，則為預設值，如果序列中有一個以上的項目，這個方法就會擲回例外狀況。 SingleOrDefault(IEnumerable, Func<TSource,Boolean>) 傳回序列中符合指定之條件的唯一一個元素，如果沒有這類元素，則為預設值，如果有一個以上的元素符合條件，這個方法就會擲回例外狀況。

Skip(IEnumerable, Int32) 略過序列中指定的項目數目，然後傳回其餘項目。

SkipLast(IEnumerable, Int32) 傳回新的可列舉集合，其包含已省略來源集合最後 count 元素的所有 source 元素。

SkipWhile(IEnumerable, Func<TSource,Boolean>) 只要指定的條件為 true，便略過序列中的項目，然後傳回其餘項目。 SkipWhile(IEnumerable, Func<TSource,Int32,Boolean>) 只要指定的條件為 true，便略過序列中的項目，然後傳回其餘項目。 項目的索引是用於述詞功能的邏輯中。

Sum(IEnumerable, Func<TSource,Decimal>) 計算在輸入序列中各項目上叫用轉換函式後所取得之 Decimal 值序列的總和。 Sum(IEnumerable, Func<TSource,Double>) 計算在輸入序列中各項目上叫用轉換函式後所取得之 Double 值序列的總和。 Sum(IEnumerable, Func<TSource,Int32>) 計算在輸入序列中各項目上叫用轉換函式後所取得之 Int32 值序列的總和。 Sum(IEnumerable, Func<TSource,Int64>) 計算在輸入序列中各項目上叫用轉換函式後所取得之 Int64 值序列的總和。 Sum(IEnumerable, Func<TSource,Nullable>) 計算在輸入序列中各項目上叫用轉換函式後所取得可為 Null 之 Decimal 值的總和。 Sum(IEnumerable, Func<TSource,Nullable>) 計算在輸入序列中各項目上叫用轉換函式後所取得可為 Null 之 Double 值的總和。 Sum(IEnumerable, Func<TSource,Nullable>) 計算在輸入序列中各項目上叫用轉換函式後所取得可為 Null 之 Int32 值的總和。 Sum(IEnumerable, Func<TSource,Nullable>) 計算在輸入序列中各項目上叫用轉換函式後所取得可為 Null 之 Int64 值的總和。 Sum(IEnumerable, Func<TSource,Nullable>) 計算在輸入序列中各項目上叫用轉換函式後所取得可為 Null 之 Single 值的總和。 Sum(IEnumerable, Func<TSource,Single>) 計算在輸入序列中各項目上叫用轉換函式後所取得之 Single 值序列的總和。

Take(IEnumerable, Int32) 從序列開頭傳回指定的連續項目數目。

TakeLast(IEnumerable, Int32) 傳回新的可列舉集合，其包含 source 的最後 count 元素。

TakeWhile(IEnumerable, Func<TSource,Boolean>) 只要指定的條件為 true，就會傳回序列中的項目。 TakeWhile(IEnumerable, Func<TSource,Int32,Boolean>) 只要指定的條件為 true，就會傳回序列中的項目。 項目的索引是用於述詞功能的邏輯中。

ToArray(IEnumerable) 從 IEnumerable 建立陣列。

ToDictionary<TSource,TKey>(IEnumerable, Func<TSource,TKey>) 根據指定的索引鍵選擇器函式，從 Dictionary<TKey,TValue> 建立 IEnumerable。 ToDictionary<TSource,TKey>(IEnumerable, Func<TSource,TKey>, IEqualityComparer) 根據指定的索引鍵選取器函式和索引鍵比較子，從 Dictionary<TKey,TValue> 建立 IEnumerable。 ToDictionary<TSource,TKey,TElement>(IEnumerable, Func<TSource,TKey>, Func<TSource,TElement>) 根據指定的索引鍵選取器和項目選取器函式，從 Dictionary<TKey,TValue> 建立 IEnumerable。 ToDictionary<TSource,TKey,TElement>(IEnumerable, Func<TSource,TKey>, Func<TSource,TElement>, IEqualityComparer) 根據指定的索引鍵選取器函式、比較子和項目選取器函式，從 Dictionary<TKey,TValue> 建立 IEnumerable。

ToHashSet(IEnumerable) 從 IEnumerable 建立 HashSet。 ToHashSet(IEnumerable, IEqualityComparer) 使用比較金鑰的 comparer 從 IEnumerable 建立 HashSet。

ToList(IEnumerable) 從 IEnumerable 建立 List。

ToLookup<TSource,TKey>(IEnumerable, Func<TSource,TKey>) 根據指定的索引鍵選擇器函式，從 Lookup<TKey,TElement> 建立 IEnumerable。 ToLookup<TSource,TKey>(IEnumerable, Func<TSource,TKey>, IEqualityComparer) 根據指定的索引鍵選取器函式和索引鍵比較子，從 Lookup<TKey,TElement> 建立 IEnumerable。 ToLookup<TSource,TKey,TElement>(IEnumerable, Func<TSource,TKey>, Func<TSource,TElement>) 根據指定的索引鍵選取器和項目選取器函式，從 Lookup<TKey,TElement> 建立 IEnumerable。 ToLookup<TSource,TKey,TElement>(IEnumerable, Func<TSource,TKey>, Func<TSource,TElement>, IEqualityComparer) 根據指定的索引鍵選取器函式、比較子和項目選取器函式，從 Lookup<TKey,TElement> 建立 IEnumerable。

Union(IEnumerable, IEnumerable) 使用預設相等比較值來比較值，以便產生兩個序列的集合等位。 Union(IEnumerable, IEnumerable, IEqualityComparer) 使用指定的 IEqualityComparer 產生兩個序列的集合等位。

Where(IEnumerable, Func<TSource,Boolean>) 根據述詞來篩選值序列。 Where(IEnumerable, Func<TSource,Int32,Boolean>) 根據述詞來篩選值序列。 述詞函式的邏輯中使用各項目的索引。

Zip<TFirst,TSecond>(IEnumerable, IEnumerable) 從兩個指定序列中的元素產生一系列元組。 Zip<TFirst,TSecond,TResult>(IEnumerable, IEnumerable, Func<TFirst,TSecond,TResult>) 將指定的函式套用至兩個序列的對應項目，產生結果的序列。

AsParallel(IEnumerable) 啟用查詢的平行化作業。 AsParallel(IEnumerable) 啟用查詢的平行化作業。

AsQueryable(IEnumerable) 將 IEnumerable 轉換成 IQueryable。 AsQueryable(IEnumerable) 將泛型 IEnumerable 轉換成泛型 IQueryable。

Ancestors(IEnumerable) 傳回包含來源集合中每個節點祖系的項目集合。 Ancestors(IEnumerable, XName) 傳回包含來源集合中每個節點祖系的已篩選項目集合。 集合中只會包含具有相符之 XName 的項目。

DescendantNodes(IEnumerable) 傳回來源集合中每個文件和項目之子代節點的集合。

Descendants(IEnumerable) 傳回包含來源集合中每個項目和文件之子代項目的項目集合。 Descendants(IEnumerable, XName) 傳回已篩選的項目集合，其中包含來源集合中每個項目和文件的子代項目。 集合中只會包含具有相符之 XName 的項目。

Elements(IEnumerable) 傳回來源集合中每個項目和文件的子項目集合。 Elements(IEnumerable, XName) 傳回來源集合中每個項目和文件的已篩選子項目集合。 集合中只會包含具有相符之 XName 的項目。

InDocumentOrder(IEnumerable) 傳回包含來源集合中所有節點的節點集合，依據文件順序來排序。

Nodes(IEnumerable) 傳回來源集合中每個文件和項目的子節點集合。

Remove(IEnumerable) 在來源集合中，從每一個節點的父節點移除這些節點。
